## âš™ï¸ Instalasi & Konfigurasi

### Prasyarat

- Node.js v18.x atau lebih baru
- MySQL Server 8.0+
- Git

### Langkah Instalasi

1. **Clone repository:**

   ```bash
   git clone https://github.com/username/suratku.git
   cd suratku
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Buat file `.env`:**

   ```env
   SESSION_SECRET=rahasia_sangat_kuat_123!
   DB_HOST=localhost
   DB_USER=root
   DB_PASS=your_mysql_password
   DB_NAME=surat_app
   PORT=50000
   ```

4. **Import database:**

   - Buat database `surat_app` di MySQL
   - Import file `database/surat_app.sql` ke database Anda

5. **Jalankan aplikasi:**

   ```bash
   node app.js
   ```

6. **Akses aplikasi:**
   Buka browser dan akses:
   ```
   http://localhost:50000
   ```

### Akun Default

Setelah import database, akun default tersedia:

- Email: `admin@example.com`
- Password: `password123`
